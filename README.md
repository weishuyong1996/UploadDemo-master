# UploadDemo
Retrofit2 文件上传及其进度显示
只涉及 Retrofit + RxJava 上传文件，像权限管理没有处理，毕竟只是演示多文件上传，显示进度

如果：只想测试一下 只需要把项目根目录 “上传文件” 目录下面的所有文件，复制到手机SD卡 “DCIM/Camera/”目录下面，即可测试文件上传

如果：不想复制文件 可以把 MainActivity 里面添加的文件改成手机里面存在的文件，不然会报 文件找不到异常
